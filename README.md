# ForkStack

ForkStack is a multi-tenant SaaS platform for restaurants, cafes, and retail vendors. It provides vendor management, QR ordering, inventory management, WhatsApp integration, subscriptions, and analytics.

## 🚀 Live Demo

**Application:** https://forkstack.onrender.com/

## 📂 GitHub Repository

[https://github.com/Ankitpatel239/forkstack]

## ✨ Features

- Multi-tenant architecture with subdomains
- Vendor dashboard
- QR code table ordering
- Inventory management
- Menu & combo management
- Offers & promo codes
- WhatsApp Business integration
- Order & payment management
- Reports & analytics

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Prisma
- PostgreSQL
- Tailwind CSS
- NextAuth
- Render
